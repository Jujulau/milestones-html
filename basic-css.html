Now let's talk about selectors!

There are:

  • type/tag selectors
  • class selectors
  • ID selectors

Type/Tag selectors

Type selectors are used to select certain HTML tags/elements.

  • For example, I could select the <i> tag and apply stylistic features to all of the <i> tags in CSS.

Now that I've added this into the CSS, all of the <i> tags will have this aqua color.

Class selectors

Class selectors are used to select multiple HTML elements on a page, and this is done through receiving the same classname. Remember the screenshots above? They both had the same 200px white border. This was accomplished through having the same classname. It saved me some time and it allowed me to apply the same effects to certain HTML elements in one go!

See how they both have the same class name?

I added this one rule to the CSS and it got applied to both images with that class!

ID selectors

ID selectors are similar to class selectors, however, they are very specific and can only be used once. This means that it only selects one HTML element, and that ID can only appear once.

For example, I used the border ID in this screenshot and I added a border to only this tag.

Descendant combinators

Descendant combinator is an element nested anywhere in an element. It could be a direct child or an element that's five levels deep.

For example the li id, one of the ui tags, and the other li tag are the descendants of the overarching parent ui tag. hope that makes sense

Basically all the HTML elements with this same tree have the same stylistic features applied to them through CSS. They all have this bique color now.
