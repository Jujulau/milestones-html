Nesting

When one element is inside another element, that is nesting. The nested element is the child element, whereas the outside element wrapping that nested element is the parent element.

For example, in the snippet, you can see that the <p> tag is the parent element and the <a> tag is the child element.

<p> and </p>

  1. <p>
    • The <p> tag is the opening tag of an element.
  2. </p>
    • The </p> tag is the closing tag of an element.
  3. The <p> tag defines a paragraph -- opening and closing tags are necessary to display that paragraph.

<a> and </a>

  1. <a>
    • The <a> tag is the opening tag of an element.
  2. </a>
    • The </a> tag is the closing tag of an element.
  3. The <a> tag defines a hyperlink -- opening and closing tags are necessary to open that link to another page.

so we got href, id, class, and target

  • id

  • class

    • description: specifies one or more classnames for an element
  • target
    • value:_blank, _self, _parent, _top
